常用的快捷键

iTerm

  • Ctrl +C 停止正在运行的程序
  • 按住command可以点击出现的链接并打开
  • 向上箭头可以找到输入过的命令
  • touch 新建一个空文件
  • sudo 获取权限

###Hexo

####一些命令

  • hexo new 生成新的 md文件的名字
  • hexo g -d生成博客页面并同步到网络
  • hexo s 在本地查看

Typora

  • command +/显示源码模式
  • 主要的语法结构和功能参照文档学习typora语法与功能

others

  • Command +T 新建标签页；

    shift+command+T恢复关掉的标签页

    电击鼠标中间的滚轮 关闭标签页

  • 快速搜索框 command +空格

  • 换输入法 Ctrl +空格

  • Ctrl + A 到行首

    Ctrl + E 到行尾

    Ctrl + B 向后

    Ctrl + F 向前

    Ctrl + N 下一行

    Ctrl + P 上一行

  • shift+command+v显示粘贴板

  • 链接到服务器 ssh zoey@o--o.win(有个公网ip)

  • 域名后台控制台（域名是在west.cn网站买的）自己添加一条cname类型的解析。这样域名就指向我的github地址。

    又从仓库的博客那个项目github pages服务中匹配上了我的域名。

    这样域名就和你的项目关联起来。

    http://iziyu.xyz

  • 测试的方法是ping一下域名能不能ping通
