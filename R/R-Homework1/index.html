基础知识检验

  • getwd() 得到工作目录

  • 下载 https://www.ncbi.nlm.nih.gov/sra?term=SRP133642 里面的 RunInfo Table 文件读入到R里面，了解这个数据框，多少列，每一列都是什么属性的元素。（参考Tips-for-Bioinformatics获取runinfo table） 这是一个单细胞转录组项目的数据，共768个细胞。

    a=read.table('SraRunTable.txt',header = T,sep = '\t',dec = '.',fill =T)#读取数据,当sep=','时候出现报错：列的数目比列的名字多；换成制表符，就可以正常读取
class(a)
[1] "data.frame"
mode(a)
[1] "list"
nrow(a)#有768行，即768个细胞
[1] 768
ncol(a)#有31列
[1] 31
for(i in 1:ncol(a)){
   print(class(a[,i]))
}#使用for循环得到数据框a每一列元素的属性
apply(a,2,function(x){
  class(x)
})#使用apply获得a中每一列的元素属性
apply(a,2,class)#使用apply获得a中每一列的元素属性的另一种方法,和上面得到相同的结果

  • 下载 https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE111229 里面的样本信息sample.csv读入到R里面，了解这个数据框，多少列，每一列都是什么属性的元素。（参考 Tips-for-Bioinformatics用R获取GEO样本信息sample.csv）

    s=read.csv('sample.csv',header = T,sep = ",",dec = '.',fill =T)#读取数据
#可以使用和上面一样的方法得到列数，以及咧里面的元素的属性。

  • 把前面两个步骤的两个表（RunInfo Table 文件，样本信息sample.csv）关联起来，使用merge函数。

    d=merge(a,S,by.x = 'Sample_Name',by.y = 'Accession')#两个表中元素相同的列进行合并

  • 对前面读取的 RunInfo Table 文件在R里面探索其MBases列；

    把前面读取的样本信息表格的样本名字根据下划线分割看第3列元素的统计情况，第三列代表该样本所在的plate，根据plate把关联到的 RunInfo Table 信息的MBases列分组检验是否有统计学显著的差异。

    分组绘制箱线图(boxplot)，频数图(hist)，以及密度图(density) 。

    使用ggplot2把上面的图进行重新绘制。

    使用ggpubr把上面的图进行重新绘制。

    随机取384个MBases信息，跟前面的两个plate的信息组合成新的数据框，第一列是分组，第二列是MBases，总共是384*3行数据。

    e=d[,c("MBases","Title")]#取出合并以后的数据中的两列Mbase和title
save(e,file = 'input.Rdata')#保存成文件

rm(list = ls())
options(stringsAsFactors = F)
load(file = 'input.Rdata')#载入数据
e[,2]#查看第二列的数据，即"Title"那一列
plate=unlist(lapply(e[,2],function(x){
  # x=e[1,2]
  x
  strsplit(as.character(x),'_')[[1]][3]
}))#取e里面第二列，即"Title"那一列，以_为分隔符拆分，取第一个元素，得到四个向量，再取第三个，就得到了该样本所在的plate，即给信息进行了分组！那个as.character很重要，不然会报错含有非字符参数。lapply函数用于遍历列表向量内的每个元素，并且使用指定函数来对其元素进行处理。返回列表向量。
table(plate)
boxplot(e[,1]~plate)#根据不同的plate分组，绘制MBases的箱线图
t.test(e[,1]~plate)#使用t-test检验分组是否有统计学显著性
e$plate=plate#在e中增加一列，为plate信息
library(ggplot2)
colnames(e)
[1] "MBases" "Title"  "plate" 
ggplot(e,aes(x=plate,y=MBases))+geom_boxplot()#使用ggplot2包画图

library(ggpubr)#使用ggpubr画图
p <- ggboxplot(e, x = "plate", y = "MBases",
               color = "plate", palette = "jco",
               add = "jitter")
# Add p-value
p + stat_compare_means(method = 't.test')
