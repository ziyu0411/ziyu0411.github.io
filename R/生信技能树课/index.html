介绍R语言以及Rstudio编辑器

  • R studio分为四区
  • 上下键找到输入过的代码
  • 帮助文档！help
  • 右上角，环境变量，运行过的变量
  • 配置文件：设置中文英文显示形式等

R语言基础变量讲解

五种变量类型(class)

向量

  • 可由c() 函数生
  • 数值，字符(需要加引号，单或双，
  • 内置函数，如LETTERS[] ，也可以取子集substr()

矩阵

  • 要经常class() 、str()
  • dim() 定义一个矩阵a的维度

数组

数据框

  • is.data.frame(), as.data.frame()
  • view()
  • 查看代码错误的问题，可以复制代码新建一个文件

列表

  • 有很多元素，每个元素可以用$符号进行取出
  • 列表里的元素可以是各种数据结构
  • lapply() 可以对每个元素进行同样的操作

数据变量索引

  • b是一个数据框，b[c(T,F,T,F),3] :就表示，一个数据框有四行三列，取出第一和三行的第三列数据。用判断的形式来取出。例如一个RNA-Seq的数据，可以用T or F来取，用grep()[大小写敏感]可以确定找到目标数据的索引

  • 关于list ,只能选元素，没有行和列。一层一层往下找到他的索引，选出的元素数据类型仍然是一个list

外部数据导入和导出

数据导入

  • 在右上角的，可以import dataset ，选择数据格式

  • read.table(' 文件路径名'，sep='\t',header=T) 以制表符分割

    具体？read.table

  • comment.char= '!' 注释字符

数据导出

  • write.csv() ，去掉行名：row.names=F 或者write.table(b,"文件名"，sep=',')
  • 可以save(b,file=)

中级变量操作

  • 函数都是有参数的！很多函数可以实现同样的目的
  • 向量化操作技巧data$V>5000就可以找到数据data中变量V的值大于5000的，会返回一系列逻辑值，用table()函数可以统计T和F各有多少，也可新建一个Ddata,挑出想要的那些数据，注意区分是按行还是按列挑；view()
  • 如何界定挑选的分界点，可以先画一个boxplot图，来根据不同的来元数据选择合适的cutoff,RNA-seq数据等
  • data$a一个=是赋值，两个==是判断

统计量

  • 每一行取平均值，rowMeans()

  • 循环，for 循环，对每一行进行平均值计算

    for(i in 1:nrow(b)){
  print(mean(as.numeric(b[i,])))
}

  • apply(b,1,)专门针对于矩阵或者数据框的？

    1代表每一行，2代表每一列，第三个参数代表了进行的函数操作，如sd求方差

    apply(b,1,function(x){
  mean(x)
})

    apply(b,1,mean)

    上面两个函数是等价的。

    apply没有for循环那么复杂，因为他是默认按照b里面的数据遗憾一行来取，计算平均值的。

  • ddd=function(x){ } 可以定义一个函数ddd实现想要的计算，一个例如计算一个数据的每一行的最大值

    rowmax=function(x){
  apply(b,1,max)
}

    就可以调用这个函数计算，但是应该含有包含该函数的

  • sample()可以随机选取一个数据中的莫几行/列，需要定义选选多少

热图

  • library(pheatmap)不同大小的数据代表不同颜色深浅

  • paste()

  • help的帮助文件里的例子，进行演习

    scale 尺度问题，相差较大的颜色会覆盖掉本来有差异但是差异较小的那些数据，显示出来就颜色无明显差别；display_numbers可以在每个小方块里显示数值

  • 选取基因表达量差异明显的矩阵绘制热图，需要的数据多，可把过程中需要的准备的变量变成function()中的参数，提供data 和grouplist以后就可以直接绘制。

id转换

  • 例如拿到一个表示差异基因的文档，批量处理

    使用R包library(stringr)处理字符型数据

    对应基因的名字？

    1. 将数据读到R里面

    2. 某个基因的名称为ENSG00000001167.13，点后面的数字是不需要的，代表着基因的更新，但主要信息是点之前的数字解释的，要去掉掉一系列的基因名字点后面的数字：

      a=strsplit('ENSG00000001167.13,'[.]') 
#用点号分割，再取他的左半部分,a是一个list
a[[1]][1]#取a里面的第一个元素，是包含两个向量，再取第一个，就是所需要的基因名字！
#必要时，使用as.character(),unlist

    对许多个，可以使用循环，但有R包解决

    library(stringr)
a$ensembl_id=str_split(a$V1,'[.]',Simplify=T)[,1] 
#str_split()这个函数可以完成对a里面V1这一列的所有数据按照"."分割，再取第一个元素就能实现对ID的简化；并将得到的ID添加到a中

    上述函数得到的是一个matrix，是因为simplify=T，否则unlist之后，类型为character，无法取列。

  • library(org.Hs.eg.db)这个包里的函数有个toTable

    关联merge()

    table()

  • 去重复!duplicated[]

  • match()按照a的顺序排序b,得有相同的id

任意基因任意癌症表达量分组的生存分析

实例需求

看一个基因在TCGA数据库里面的保存的各个癌症里面的生存关系图

  • 网页工具搜索找到基因：在OncoLnc中搜索ARHGAP18这个基因，会给出这个基因的cox回归的结果，自己定义一个低表达和高表达的数值，网页工具汇给处一个Kaplan plot

  • 可将该数据下载（表达矩阵）下来使用R语言重现

    rm(list = ls())##魔幻操作，一键清空
options(stringsAsFactors = F)
a=read.table(file ='/Users/zoey/BLCA_93663_50_50.csv',header = T,sep = ',')#读取csv
view(a)#查看读取后的数据状态(也可点击右侧的结果查看)
colnames(a)#查看列名字有哪些
data=a
library(ggstatsplot)#载入使用的R包,第一次使用应该事先安装install.packages()
ggbetweenstats(data = dat,x=Group,y=Expression)#会得到一个关于表达量高和低的小提琴图，是按照Group分组的

    对数据生存分析

    library(ggplot2)
library(survival)
install.packages("survminer")
library(survminer)#安装和载入所需要的R包
table(dat$Status)#查看STATUS这一列不同标签的数值，dead 125，alive 385
dat$Status=ifelse(dat$Status=='Dead',1,0)
sfit <- survfit(Surv(Days,Status)~Group,data=dat)
sfit#
summary(sfit)
ggsurvplot(sfit,conf.int = F,pval = TRUE)#可以画出生存分析图(如下图)
ggsave('survival_ARHGAP18_in_BLCA.png')#保存图片


ggsurvplot(sfit,palette = c("#E7B800","#2E9FDF"),
           risk.table = TRUE,
           pval = TRUE,
           conf.int = TRUE,
           xlab="Time in months",
           ggtheme = theme_light(),
           ncensor.plot = TRUE) #对该函数根据需要进行参数微调以后，可以得到想要的与网页工具不同的图片如下图
ggsave('survival_ARHGAP18_in_BLCA.png')

最重要的是 查看函数ggsurvplot()的帮助文档,了解这些参数的用途

b-rnorm(10)

    rm(list = ls())##魔幻操作，一键清空
options(stringsAsFactors = F)
source("http://bioconductor.org/biocLite.R")##安装BiocInstaller
biocLite("airway")
install.packages(airway)#安装airway,bioconductor的数据包
library(airway)#载入数据包
data(airway)#用data函数
exprSet=assay(airway)

NOTE:
我在载入R包airway的时候，就按照教程直接library()，但未发现，我以为是没安装，就想采用了install.packages()来安装包，但是并不能成功，R提示找不到对象”airway”。

通过搜索发现，它是bioconductor里的数据包，bioconductor分为三个包，功能函数包、数据包和注释包 ，注释包是各种各样的基因、芯片间的转换；数据包的功能就是为你加载数据，需要使用data()函数把它加载到当前的工作空间中。

要使用airway,需要安装bioconductor,详情在R-Tips-R包安装,这篇文章中写到了四种常见的R包安装方式。
